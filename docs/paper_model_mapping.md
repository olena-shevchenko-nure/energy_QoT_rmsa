# Paper Model Mapping

This file maps the names used in the paper to the internal policy identifiers, artifact folders, and runtime behavior used by the evaluation configuration.

| Paper name | Runtime policy id | Artifact folder | Notes |
|---|---|---|---|
| Calibrated DQN-Override baseline ranker | `torch_dqn_candidate_ranker_distill_old10` | `artifacts/models/calibrated_dqn_override` | DQN-like MLP ranker with calibrated override-rate behavior over the Energy-Aware-KSP-BM-FF baseline candidate. |
| LightGBM-override ranker | `lightgbm_candidate_ranker_old10` | `artifacts/models/lightgbm_override_old10` | LightGBM candidate ranker exported for runtime override behavior. The historical suffix `old10` is preserved because it identifies the experiment lineage. |
| GNN-CNN Full DQN Ranker | `gnn_cnn_dqn` | `artifacts/models/full_dqn_stratified32_e5` | Full neural candidate ranker using graph/spectrum/request/action encoders and a DQN-style Q head. |
| XLRON Counterfactual Ranker | `top32_xlron_stabilized_ppo` | `artifacts/models/xlron_cf_rank_g160_bucket_guard` | XLRON-style policy/ranker fine-tuned with counterfactual ranking signal and bucket guard. |
| A3C Policy Distilled from Full DQN | `gnn_cnn_a3c` | `artifacts/models/a3c_distill_full_dqn` | Actor-critic model distilled from the Full DQN teacher. |
| Energy-Aware-KSP-BM-FF | `energy-aware-ksp-bm-ff` | none | Deterministic heuristic implemented in source code. |
| KSP-FF | `ksp-ff` | none | Deterministic K-shortest-path first-fit heuristic. |
| KSP-BM-FF | `ksp-bm-ff` | none | Deterministic K-shortest-path best-modulation first-fit heuristic. |

The paper comparison uses the shared Top-N candidate surface rather than unconstrained route/slot generation.
Learned models and deterministic heuristics therefore operate over feasible candidate actions generated by the same data/runtime pipeline.
